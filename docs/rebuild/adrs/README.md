# Architecture Decision Records

This folder stores architecture decisions for the rebuild.

## ADR Rules

- Use one ADR per meaningful decision.
- Keep rationale and alternatives concise and explicit.
- Include a rollback/reversibility note where relevant.
- Never modify historical ADR decisions silently; supersede with a new ADR.
